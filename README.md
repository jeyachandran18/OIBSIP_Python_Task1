## üìå Objective
To create a simple **BMI (Body Mass Index) Calculator** using Python‚Äôs `turtle` graphics module that demonstrates basic GUI input, calculations, and visual output for beginners learning turtle programming.

## ‚öôÔ∏è Steps Performed
1. Imported the `turtle` module and initialized the screen with a title.
2. Used `screen.numinput()` pop-up dialogs to collect user weight (kg) and height (m).
3. Calculated BMI using the formula:  
   **BMI = weight / (height ** 2)**
4. Applied `if-elif-else` conditions to classify BMI into:
   - Underweight
   - Normal
   - Overweight
   - Obese
5. Displayed the BMI value and health category using `pen.write()` at specific coordinates.
6. Added `exitonclick()` to close the turtle window on user interaction.
